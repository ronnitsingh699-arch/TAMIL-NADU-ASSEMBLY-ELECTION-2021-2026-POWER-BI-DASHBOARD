# 🗳️ Tamil Nadu Assembly Election Analysis (2021 vs 2026) | Power BI Dashboard

## 📌 Project Overview

This Power BI dashboard provides a comparative analysis of the **Tamil Nadu Legislative Assembly Elections (2021 vs 2026)**. It enables users to explore changes in party performance, vote share, seat distribution, constituency-wise trends, and voter participation through interactive visualizations.

The dashboard is designed to help users understand electoral shifts, identify emerging political patterns, and compare election outcomes between the two election years. Election results and statistics are based on official Election Commission data. :contentReference[oaicite:0]{index=0}

---

## 🎯 Objectives

- Compare the 2021 and 2026 Tamil Nadu Assembly election results
- Analyze seat distribution across political parties
- Compare vote share between election years
- Explore constituency-wise winning parties
- Identify regional political trends
- Provide interactive filtering for better analysis
